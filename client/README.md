# Frontend Client

React application for the Transcript Helper voice-enabled AI chat assistant.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open in your browser at `http://localhost:3000`

## Features (To be implemented)

- Voice transcription using Web Speech API
- Chat interface with message bubbles
- Cut/Discard controls
- Camera integration
- Persistent conversation history

## Browser Requirements

- Chrome/Edge recommended for Web Speech API support
- HTTPS or localhost required for microphone/camera access
- Modern browser with ES6+ support
